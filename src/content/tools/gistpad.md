- Create pastes: unlimited public and unlisted pastes with instant permalinks.
- One‑click sharing: copy link and get raw/embed links without an account.
- Syntax highlighting: automatic detection for common languages and plain text.
- Expiration and visibility: set paste expiration (short presets) and choose Public/Unlisted/Private (private requires account).
- Basic versioning: lightweight revisions (limited history depth per paste).
- Raw and embed modes: raw text endpoint and embeddable snippet iframe.
- Download and copy: download paste as .txt or copy to clipboard.
- Minimal editor: fast, keyboard-friendly editor with common shortcuts and paste import.
- Search and tagging: basic site search and tag-based discovery for your own public/unlisted pastes.
- Mobile/responsive UI: full editor and viewer functionality on mobile and desktop.
- Client integrations: basic webhook support and simple API endpoints with rate limits suitable for lightweight automation.
- Privacy defaults: sensible defaults (unlisted by default for non-public flows) and optional password protection for single pastes.
- Community support and docs: public docs, examples, and email/community support for free users.
